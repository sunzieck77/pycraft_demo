int - Integer คือ ข้อมูลชนิดตัวเลขจำนวนเต็ม
ตัวอย่าง
x = -10
apple = 12
print(20)
float - Float คือ ข้อมูลชนิดตัวเลขทศนิยม
ตัวอย่าง
x = -10.2
carrot = 12.5
print(3.14)
str - String คือ ข้อมูลชนิดตัวอักษรข้อความ
ตัวอย่าง
x = "Hello"
words = "How are you"
print("Hello Python!")
bool - Boolean คือ ข้อมูลชนิดค่าความจริงเท็จ
ตัวอย่าง
x = True
light_status = False
print(True)
Variable - ตัวแปร เปรียบเหมือนกล่องเปล่า ที่สามารถเก็บข้อมูลใดได้
ตัวอย่าง
ชื่อตัวแปร  = ค่าข้อมูล
ตั้งชื่อตัวแปร ตามด้วยเครื่องหมายเท่ากับ เป็นการบอกว่าจะกำหนดค่าข้อมูลอะไร แล้วจึงตามด้วยข้อมูลที่ต้องการ

age = 16
number = 32
print(64)
print() - ปริ้นต์ คำสั่งแสดงผล
ตัวอย่าง
print("ข้อความที่ต้องการ")
ใช้คำสั่ง print(" ") ข้อความที่ต้องการจะอยู่ในเครื่องหมายฟันหนูหรืออัญประกาศ หรือสามารถปริ้นต์แต่ละชนิดข้อมูลได้ดังนี้

#ปริ้นข้อมูลชนิดข้อความ
print("Hello Mr.Pieck")

#ปริ้นข้อมูลชนิดตัวเลขจำนวนเต็ม
print(22)

#ปริ้นข้อมูลชนิดตัวเลขทศนิยม
print(63.5)

#ปริ้นข้อมูลชนิดค่าความจริงเท็จ
print(True)
input() - อินพุต คำสั่งรับข้อมูลจากผู้ใช้
ตัวอย่าง
age = input("Input your age: ")
สร้างตัวแปรมารับข้อมูลก่อนและชื่อว่า age จากนั้นใช้เครื่องหมายเท่ากับเป็นการกำหนดค่าข้อมูล ตามด้วยคำสั่งรับข้อมูล input("") สิ่งที่อยู่ในเครื่องหมายอัญประกาศจะเป็นข้อความที่ต้องการบอกให้ผู้ใช้กรอกข้อมูลอะไร

number = input("Input number: ") print(number)
